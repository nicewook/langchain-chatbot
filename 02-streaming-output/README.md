# Chat with memory

Base: LangChain, OpenAI, Memory를 활용한 챗봇 구현

## 실행

```bash
python3 02.py
```

## 추가 구현
- 입력을 받는 라이브러리 사용: `prompt_toolkit`
- 텍스트에 색상 추가: `rich`
- 답변을 스트리밍으로 출력하도록 함

## install
```bash
pip install rich
pip install prompt_toolkit
```
